# CloveLib Studios

Documentation website for CloveLib plugins built with React + TypeScript + Vite.

[![Build Site](https://github.com/CloveLib/Studios/actions/workflows/build.yml/badge.svg)](https://github.com/CloveLib/Studios/actions/workflows/build.yml)
[![CodeFactor](https://www.codefactor.io/repository/github/clovelib/studios/badge)](https://www.codefactor.io/repository/github/clovelib/studios)

## Live Site

[clovelib.win](https://www.clovelib.win)

## Quick Start

```bash
# Install
npm install

# Dev server
npm run dev

# Build
npm run build

# Preview
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
